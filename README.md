# Buzzshock Roulette
## Customizable [Multishock/Pishock integration](https://mshock.akiradev.me/) for Buckshot Roulette

Made on Buckshot Roulette 2.2.0.6, built for [BRML-E](https://github.com/D1GQ/BuckshotRouletteModLoaderExtended) 1.0.2.
Made this in 2 days after having 0 knowledge on scripting in Godot for fun. Not much else to it.

* Features:
  * Granular event types (All Shots and All Double or Nothing items supported, including each outcome for Expired Medicine)
  * Config in human readable JSON
